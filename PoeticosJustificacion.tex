\section*{Explicación de cantos poéticos}

Para la selección de esta colección, deberemos de definir primeramente que es un canto poético.
Un canto poético es una pieza que cumple con la menos una de las siguientes condiciones:
\begin{itemize}
    \item Usa alguna metáfora que no es bíblica
    \begin{itemize}
        \item Ser como un pájaro herido
    \end{itemize}
    \item Relata una experiencia personal (testimonio)
    \item Expresa una creencia, sentir o sensación
    \begin{itemize}
              \item Sentir la presencia de Dios sensorialmente
    \end{itemize}
    \item Imagina a partir de un pasaje de la biblia o alguna experiencia mística
    \begin{itemize}
              \item Jesucristo invitándote a pescar
    \end{itemize}
    \item Usa algún concepto que no es bíblico, aunque pueda ser parte de la cultura local
    \begin{itemize}
        \item Ofrenda de amor
        \item Fiesta para Dios
        \item Río del Espíritu Santo
    \end{itemize}
    \item Hace algún compromiso, declaración o voto
    \begin{itemize}
        \item Vivo para adorarte
        \item Todo lo que gane será para Tí
        \item Por Tí vivo
        \item Tu mirada me hace sentir paz
    \end{itemize}
\end{itemize}
